Airship data

Telemetry recovery system for high-altitude balloons

Airship Data is an all-in-one solution for live data recovery from high-altitude balloons in hobby and educational projects. The package is comprised of a desktop application and a lightweight mobile app. The user Installs the app on any Android smartphone and launches it with the payload. The app will send down to the desktop application live data, comprised of GPS coordinates, photos, speed, gyroscope and temperature, all the while it stores high resolution video for later recovery. After landing, the ringer feature can be used to quickly find the payload’s exact location.

While this was made for an internal marketing campaign, we can license the software to anyone who dreams of sending a high-altitude helium filled balloon to the outer edge of the atmosphere. You can contact us for more info. We will deliver Airship Data custom fitted to your needs.
